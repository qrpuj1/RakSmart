## 线路选择：大陆优化、CN2、国际BGP有啥区别

聊了这么多配置和价格，还有一个很重要的问题：线路怎么选？RakSmart提供多种线路，价格差距挺大，搞不清楚的话容易选错。

**大陆优化线路**

这是最基础的优化线路，主要接入电信163骨干网、移动CMI直连，再加上一些国际线路如HE、GTT、TELIA、NTT等。整体访问速度比普通国际线路好，但在晚高峰时段可能会有一定的拥堵。价格相对便宜，适合预算有限但又希望有一定访问速度的用户。

**CN2线路**

CN2是中国电信推出的精品网络，分为CN2 GT和CN2 GIA两种。CN2 GT是半程CN2，回国走CN2线路，去程走普通线路。CN2 GIA是全程CN2，去程和回程都走CN2，访问速度和稳定性都是顶级的。
